## Publication

Le site est publié automatiquement sur **GitHub Pages** à chaque mise à jour de la branche `main`
(voir [`.github/workflows/deploy-pages.yml`](./.github/workflows/deploy-pages.yml)).

Le fichier `index.html` est autonome : tout le CSS est en ligne, aucune dépendance externe
(la police Inter est chargée depuis Google Fonts avec repli sur les polices système).
